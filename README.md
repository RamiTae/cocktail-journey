# coctail-journey

`칵테일저니`는 개인이 마신 칵테일의 정보와 그에 대한 별점, 맛 평가를 간단하게 정리할 수 있는 기록 앱입니다.

사용 API: https://www.thecocktaildb.com/

---

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
